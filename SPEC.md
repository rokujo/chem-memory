## 1. 単元の構造

「**試薬 × 金属イオン**」のクロス表(高校化学資料集の代表的な表)をベースにしている。元ネタは数研「化学」教科書、リードA問題集、写真の資料集。

ただし**写真の表をそのままWebに焼き直してはいない**。スマホで横幅が取れない制約から、表ではなく「**反応データを縦に展開する**」アプローチを取った。

### 含めるもの

- 主要金属イオン10種(Ag⁺, Pb²⁺, Cu²⁺, Fe²⁺, Fe³⁺, Al³⁺, Zn²⁺, Ca²⁺, Ba²⁺, Na⁺, K⁺)
- 主要試薬10種(Cl⁻, S²⁻酸性/塩基性, NH₃ 少量/過剰, NaOH 少量/過剰, CO₃²⁻, SO₄²⁻, CrO₄²⁻)

### 別単元に分離するもの (この単元には入れない)

- **錯イオン形成** ([Cu(NH₃)₄]²⁺ 等の組成式・色暗記) → 別単元
- **炎色反応** → 別単元
- **酸化還元の半反応式** (MnO₄⁻, Cr₂O₇²⁻ 等) → 別単元(v2で実装済み)
- **[Fe(CN)₆]³⁻/⁴⁻ の血赤色・濃青色反応** → Fe検出単元(将来追加)

「沈殿反応」と「錯イオン形成」は連続した現象だが、覚え方の論点が違う(両性 vs 錯イオン形成の区別)ので分離した。ただし**沈殿が過剰量で錯イオンになる流れ**(Cu²⁺ + NH₃: 青白沈殿 → 深青色錯イオン)は段階反応として本単元内で表現する(下記参照)。

## 2. データ構造の設計判断

### フラットなReactionリスト

`ion × reagent` のマトリックス型ではなく、各反応を1レコードにしたフラットな配列で持つ。

```typescript
type Reaction = {
  ion_id: string;
  reagent_id: string;
  result: 'precipitate' | 'complex' | 'no_reaction' | 'redissolve' | 'colored_solution' | 'out_of_scope';
  color?: string;
  product?: string;
  note?: string;
  sequence_group?: string;   // 段階反応のグループID
  sequence_order?: number;   // グループ内の順序
};
```

**フラットにした理由**: クイズ実装時に「沈殿色だけ抽出」「両性元素だけ抽出」のようなクエリがしやすい。マトリックス型だと「副条件(酸性/過剰)」が表現しにくい。

### result列挙型の6種類

- `precipitate`: 沈殿が生じる
- `complex`: 錯イオン形成で溶解 (NH₃過剰など)
- `redissolve`: 両性水酸化物の再溶解 (NaOH過剰の Al, Zn, Pb)
- `no_reaction`: 反応しない/沈殿しない
- `colored_solution`: 着色溶液(沈殿しないが色がつく、現状未使用)
- `out_of_scope`: 高校範囲外

`complex` と `redissolve` を分けているのは、**メカニズムが違うから**(配位子による錯イオン vs 両性水酸化物の再溶解)。教育的に区別する価値がある。

### 段階反応の表現 (sequence_group)

写真の資料集では一つのセルに「少量で白沈殿、過剰で錯イオン溶解」と上下二段で書かれている。これをデータで表現する方法を3案検討:

- 案A: 一つの reaction レコードに「少量の結果」と「過剰の結果」を両方持つ
- 案B: 別の試薬(NH₃少量とNH₃過剰)として独立した行に分ける
- **案C(採用): 別レコードにして sequence_group で紐付ける**

案Cを採用した理由:
- 各「条件下での反応」を独立したクイズ問題にできる
- 段階反応の表示(矢印つき)はUI側で sequence_group をルックアップして組み立てる
- マトリックス表示でも「NH₃少量」「NH₃過剰」を別行として表示できる

```typescript
// 例: Cu²⁺ + NH₃ の段階反応
{ ion_id: 'cu', reagent_id: 'nh3_few', result: 'precipitate', color: '青白色', product: 'Cu(OH)₂',
  sequence_group: 'cu_nh3', sequence_order: 1 },
{ ion_id: 'cu', reagent_id: 'nh3_ex',  result: 'complex', color: '深青色', product: '[Cu(NH₃)₄]²⁺',
  sequence_group: 'cu_nh3', sequence_order: 2 },
```

イオン中心ビューでは `getSequenceFor()` で段階反応を取得し、起点(order=1)のみ `SequenceReactionRow` で描画。「青白沈殿 → 深青色溶液」と矢印で時系列表示する。

## 3. UI設計の三ビュー構造

写真の資料集のような巨大マトリックスはスマホ画面で見るのが困難。そこで「**用途別の3ビュー**」を提供。

### イオン中心ビュー (IonView)

- 一つの金属イオンを選ぶと、そのイオンに対する全試薬の反応が縦に並ぶ
- 「銅についての全知識」を一画面で把握する用途
- 上部にイオン式タブ(横スクロール可)、下部に試薬×反応のリスト
- 段階反応は矢印つきで1行に圧縮表示

### 試薬中心ビュー (ReagentView)

- 一つの試薬を選ぶと、全イオンとの反応が一覧
- 「この試薬で何が沈殿するか」を見る用途
- 「沈殿・反応するイオン」「反応しないイオン」「高校範囲外」の3グループに分けて表示
- 「沈殿しない」イオンを別グループにまとめて省スペース化

### 俯瞰マトリックス (MatrixView)

- 資料集と同じ全体表を横スクロール可能にレンダリング
- 試薬列(左)が sticky で固定される
- 各セルは色見本+略色名(「青白」など)を表示
- 「全部見たい」生徒向けの参照ビュー(暗記の主役ではない)

## 4. クイズ設計

### 共通方針

- スコアは「正答 X / Y (Z%)」をミニ表示。リセットは画面遷移で自然発生
- 解答中はノーヒント、解答後に化学式・色見本・解説を表示
- 「次の問題」ボタンで連続出題

### クイズ1: イオン中心 (4択)

- 出題: 「Cu²⁺ + NH₃ aq(少量) → ?」のような形で組合せに対する現象を当てる
- 選択肢: 色見本+色名+状態(沈殿/錯イオン溶解/再溶解/沈殿しない)
- ダミー作成: 同じ試薬の他の結果からランダム3つ + 「沈殿しない」を必ず混ぜる

**重要な設計判断 (v8で修正)**:
ダミーの一意性判定は「画面表示部分(色名+状態)のみ」で行う。化学式は判定キーに含めない。これは、たとえば S²⁻酸性で「黒色沈殿」が複数イオン(CuS, Ag₂S, PbS)で発生するため、化学式まで判定キーに含めると表示上「黒色沈殿」が複数並んでしまうのを防ぐため。

ダミー候補が同試薬で3個確保できない場合(例: S²⁻酸性は黒沈殿 or 沈殿しないの2択になる)、**他試薬の結果からも借用**する。教育的に少し不自然(その試薬では起きない色を選択肢に並べる)だが、選択肢の難易度を確保するため許容している。

### クイズ2: 試薬中心 (複数選択)

- 出題: 「S²⁻塩基性で**黒色**沈殿を作るイオンを**すべて**選んでください」
- 候補: その試薬で何らかの反応(範囲外を除く)があるイオンのみ
- 正解判定: 完全一致(漏れ・余分どちらも不正解)
- 答え合わせ後の表示: 正解のイオンは緑、選んで間違えたのは赤、選び忘れは薄緑+「選び忘れ」表示

候補イオンを絞る理由: Na⁺ や K⁺ をいちいち候補に並べると「全部いいえ」が増えて単調になるため。

### クイズ3: 色クイズ (複数選択)

- 出題: 「**白色**の沈殿を作る組合せを**すべて**選んでください」
- 候補: 正解2〜3個 + ダミー4〜5個 = 計6〜8個

**重要な設計判断 (v5で修正)**:
当初は候補12個程度で出題していたが「難しすぎる」とのフィードバックを受け、選択肢を絞った(案A)。正解ペアが多い色(白沈殿は11個以上)も、ランダムサンプリングで3個に絞って出題する。

`MAX_CORRECT_PAIRS = 3`, `NUM_DISTRACTORS = 5` で調整可能。

## 5. 色表現 (COLOR_PALETTE)

化学の色は教科書記述と微妙に違うことが多い。実装時に苦労した点を記録:

- **「無色」は特別扱い**: 透明を表現するため市松模様のCSSグラデーション(`ChipSwatch` 内で分岐)
- **白色** (`#fafafa`): 必ず境界線を描く。背景同色だと埋もれるため
- **青白色** (`#a6c8e8`): Cu(OH)₂ の淡い青み。白色とは明確に異なる必要あり
- **赤褐色** (`#7c2d12`): Fe(OH)₃ の赤錆色。これは焦げ茶寄りの赤(初期値 `#b91c1c` だと灰色っぽく見えた)
- **褐色** (`#451a03`): Ag₂O の黒褐色。より黒寄り
- **緑白色** (`#bbf7d0`): Fe(OH)₂ の沈殿色

**色名と物質の対応**は教科書ごとに揺れがある。Fe²⁺ + NH₃ の沈殿は「緑白色」(数研)で実装しているが、資料集によっては「黒」「灰緑」と書かれることもある。**勝手に修正せず、前田氏に確認**してから変える。

## 6. 既知の問題と未解決の論点

### コンテンツ要レビュー

以下は前田氏が資料集と照合して確認する予定:

- Fe²⁺ の NH₃少量・NaOH少量での沈殿色(「緑白色 Fe(OH)₂」で実装、ただし資料集の表では「黒」と読める表記がある)
- Fe³⁺ + S²⁻塩基性 の黒沈殿の解釈(「Fe³⁺ → Fe²⁺ に還元されて FeS が沈殿」と注釈付き)
- Co²⁺, Ni²⁺, Mn²⁺, Cr³⁺ などの遷移金属を含めるかどうか(現状はメイン10種のみ)

### クイズ1の他試薬借用問題

クイズ1で同試薬のダミーが不足する場合、他試薬の結果から借用する仕組みになっている。教育的にやや不自然だが、選択肢の難易度確保のため許容。気になるなら以下の対策が考えられる:

- 同試薬で多様な結果がある問題のみ出題する(出題範囲が狭まる)
- 借用したダミーには「(参考)」マークをつける(視覚的にうるさい)

### 化学式の表示タイミング

クイズ1と3では「化学式は解答時のみ表示」(ヒントにならないため)。
クイズ2では化学式は元から答え合わせ後のみ。

ただし**学習途中で「化学式は完璧だけど色を覚えていない」生徒には、色クイズの選択肢に化学式が見えたら答えがバレる可能性がある**。これは将来「化学式を隠す上級モード」を追加する余地がある。
